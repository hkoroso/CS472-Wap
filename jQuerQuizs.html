1. In jQuery terminology, what is the difference between the “jQuery function” and the “jQuery object”?
    the jquery library define a single global function named jQuery(), the global symbol $ is a shortcut for it. the value return by this function represent a set of zero or more DOM elements and is known as a jQuery object.
    -jQuery functionthe global jQuery function that is normally aliased as $ for brevity
    - jQuery objectthe object returned by the jQuery function that often represents a group of elements
  2. Write jQuery code to find all h1 elements that are children of a div element and make their background color red. 
    <script src="https://ajax.googleapis.com/ajax/libs/jquery/3.6.3/jquery.min.js"></script>
<script>
    $(document).ready(function(){
        $('div > h1').css("background", "red"); 
    });
</script>
